CarVilla - Free Bootstrap Car Dealer Template 🚗

CarVilla is a sleek and modern one-page Bootstrap 3-based Car Dealer Automotive website template, designed for automotive businesses, car dealers, and vehicle marketplaces.

🚀 Features
- Fully responsive and mobile-friendly
- Clean and professional design
- Bootstrap 3 framework
- jQuery-powered interactivity
- Smooth scrolling and animations
- Sticky navigation bar
- Optimized for performance
